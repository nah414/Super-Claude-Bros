## Task 2: Cannon + BulletBill (TDD)

- [ ] **tests/test_bullet.py**:
```python
import pygame
from game import settings as S
from game.entities.bullet import BulletBill
from game.entities.cannon import Cannon

def test_bullet_flies_straight_and_despawns():
    lvl = type("L", (), {"solids": [], "width_px": 2000})()
    b = BulletBill(100, 100, 1)
    x0 = b.x
    for _ in range(5):
        b.update(lvl)
    assert b.x > x0 and b.y == 100              # straight, no gravity
    for _ in range(S.BULLET_LIFE):
        b.update(lvl)
    assert not b.alive

def test_cannon_fires_toward_in_range_player_on_cooldown():
    c = Cannon(500, 300)
    # too close -> no fire
    assert c.tick(player_cx=520) is None
    c.timer = 0
    shot = c.tick(player_cx=500 + S.BULLET_MIN_DIST + 50)   # in range
    assert shot in (1, -1)
    assert c.timer == S.BULLET_COOLDOWN                       # reset
```
- [ ] **game/entities/bullet.py** ÔÇö `BulletBill(x, y, direction)`: `__init__` centers a `BULLET_SIZE` box at (x,y); `score=BULLET_SCORE`; `update(level)` Ôćĺ `x += BULLET_SPEED*dir`, `life -= 1`, die if `life<=0` or `x` outside `[-50, level.width_px+50]`; `draw` Ôćĺ `assets.draw_bullet`.
